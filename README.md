# Exercícios de HTML e CSS

Este repositório reúne meus exercícios de **HTML e CSS**, desenvolvidos ao longo dos meus estudos em desenvolvimento web.

A ideia aqui é praticar bastante a estruturação de páginas, estilização, organização de conteúdo e evoluir passo a passo na construção de interfaces.

Ao longo dos exercícios trabalho conteúdos como:

- Estrutura básica do HTML  
- Títulos, parágrafos e formatação de texto  
- Listas e links  
- Imagens  
- Tabelas  
- Organização de layout  
- Estilização com CSS  

Esse repositório também serve como um registro da minha evolução no HTML e CSS 🚀

## ▶ Como executar os exercícios

### 🔹 Usando Live Server

1. Abra o projeto no **VS Code**  
2. Clique com o botão direito no arquivo `index.html`  
3. Selecione **"Open with Live Server"**  

O navegador abrirá automaticamente com a página funcionando.

### 🔹 Abrindo manualmente

Basta dar dois cliques no arquivo `.html` para abrir no navegador.
